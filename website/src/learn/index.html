Learn Houston

The platform for AI-native products.

An AI-native workspace is an app where the AI can take every action a human can take. The human can take every action the AI can take. Neither is a guest. Houston is the platform for building them. This guide explains the concept and how to build your first agent.

Start reading Skip to building

Who this is for

Founders and developers who want to build AI-native products on Houston. Whether you're defining a simple agent or building custom React components, this guide gets you there.
